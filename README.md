## Description

In task list we can open another site with react-router to display details of task, also tasks can be marked as done or removed from list with buttons. If we want we can add example tasks to the list.

#### Used technologies: 
- ES6
- React.js
- React Router
- Redux
- Styled Components
- Flexbox
- Grid
- Fetch
